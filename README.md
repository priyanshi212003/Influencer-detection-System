# Influencer-detection-System
The Influencer Detection System is a web application that suggests relevant YouTubers based on user-defined filters, including category, country, and follower count. It features interactive visualizations to help users analyze influencer performance and engagement metrics, facilitating data-driven decisions for brand collaborations.
In today’s digital landscape, finding the right influencer can be challenging, given the vast number of content creators. This system simplifies that process by allowing users to input specific filters, such as category, country, and follower count. The application then utilizes a sophisticated algorithm to suggest the most suitable YouTubers, ensuring that users find influencers who align with their marketing goals.
Features :
Customized Influencer Suggestions: Users can easily input criteria to receive tailored recommendations of YouTubers. This personalization helps in selecting influencers who resonate with the target audience.
Dynamic Filtering System: The application offers various filtering options that enable users to narrow their search based on:
Category: Content type (e.g., beauty, technology, travel)
Country: Geographical location of the influencer
Follower Count: Ensures suggestions are relevant to the user’s outreach goals.
Interactive Visualizations: The system presents various visualizations, such as:
Engagement Metrics: Analyzing likes, comments, and shares to assess influencer effectiveness.
Trends: Understanding the popularity of influencers over time.
Demographic Insights: Visual data representing audience characteristics, aiding users in making informed choices.
Technologies Used :
The Influencer Detection System is built using modern technologies, including:
Python: For backend development and data processing
Pandas: To handle data manipulation and analysis
Seaborn & Matplotlib: For creating insightful visualizations
Streamlit: To develop an interactive web interface
